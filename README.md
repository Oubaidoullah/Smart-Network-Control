# Wireless

Wireless is a cross-platform network management and diagnostic suite designed to provide advanced control and visibility over wireless connectivity across multiple environments. It supports **Android, Windows, macOS, and Linux**, offering a consistent set of tools for network professionals and power users alike.

## Multi-Platform Support

Wireless is built to be truly cross-platform, ensuring you have the same powerful tools whether you are on your mobile device or your desktop workstation:
- **Android**: Full-featured mobile app with a native experience.
- **Desktop (Windows, macOS, Linux)**: Robust desktop implementations for deep system-level network management.

## Key Features

- **Network Diagnostics**: Detailed analysis of network health, latency, and packet information.
- **WiFi & Bluetooth Control**: Unified interface for scanning, connecting, and managing wireless interfaces.
- **Setup Wizard**: Guided configuration to get your environment ready on any platform.
- **Saved Networks & Profiles**: Synchronized management of known networks and hardware configurations.
- **Web Integration**: Seamlessly sync with the [i-Wireless Web Dashboard](https://i-wireless.web.app/) for centralized management.
- **System Security**: Integrated security protocols (powered by the SystemSecurity module) to ensure safe connectivity.

## Technical Details

- **Core Languages**: Kotlin, Scala
- **Build System**: Gradle
- **Platforms**: Android (API 26+), Windows, macOS, Linux
- **Security**: Custom security layer for encrypted communication and safe network handling.
